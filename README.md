# 坦克大战 🎮 Battle City

经典 **Battle City**（坦克大战）风格的 HTML5 游戏。使用 Canvas 实现，纯前端，无需任何外部依赖。

## 游戏玩法

- **目标**：消灭所有敌方坦克，保护己方基地（底部的鹰/旗帜）
- **20~30 辆敌军坦克**逐波出现，全部消灭即可过关
- 每关有 **3 条命**，基地被摧毁或命用完则游戏结束

## 操作方式

| 按键 | 动作 |
|------|------|
| ↑ ↓ ← → | 移动坦克 |
| 空格键 | 射击 |
| Enter | 开始游戏 / 重新开始 |
| P | 暂停 / 继续 |

## 游戏特性

- 🧱 **可破坏的砖墙** — 子弹击中后消失
- ⬜ **不可破坏的钢墙** — 子弹击中后弹开
- 🏠 **基地保护** — 必须在敌方的围攻中守护基地
- 🤖 **AI 敌人** — 具有寻路逻辑（趋向玩家或基地）
- 💥 **爆炸特效** — 粒子效果带来更好视觉反馈
- 📊 **实时状态面板** — 分数、生命、剩余敌人一目了然
- 🎯 **多种敌人类型**：普通、快速、重甲

## 关卡

当前内置 **3 个关卡**，难度递增：
- **第 1 关**：20 辆敌人，布局经典
- **第 2 关**：25 辆敌人，更开阔，移动战
- **第 3 关**：30 辆敌人，迷宫布局，高强度

## 本地运行

直接用浏览器打开 `index.html` 即可游玩：

```bash
# 克隆仓库
git clone https://github.com/YOUR_USERNAME/tank-battle.git
cd tank-battle
# 直接打开
start index.html    # Windows
open index.html     # macOS
```

或使用本地服务器：

```bash
# Python
python -m http.server 8000
# 然后访问 http://localhost:8000

# Node.js
npx serve .
```

## 技术栈

- **HTML5 Canvas** — 游戏渲染
- **原生 JavaScript (ES6+)** — 游戏逻辑
- **CSS3** — UI 样式与布局

零依赖，单文件，开箱即玩。
